Complete Setup Guide

Step 1 Download Dataset

What to do: Download the Online Retail dataset from Kaggle

  1. Visit Kaggle Online Retail Dataset
  2. Click "Download" button
  3. Extract the ZIP file
  4. Find OnlineRetail.csv
  5. Copy to data/ folder in your project
File Details: OnlineRetail.csv (~21 MB) contains 541,909 transactions
Step 2 Install Python Packages

What to do: Install required Python packages

Open Command Prompt/PowerShell in your project folder and run:

pip install -r requirements.txt

This installs:

  • pandas (data manipulation)
  • numpy (numerical operations)
  • matplotlib & seaborn (visualization)
  • Flask (web framework)
Step 3 Start Web Application

What to do: Start the Flask web application

In Command Prompt/PowerShell, run:

python app.py

You should see:

* Running on http://localhost:5000
* Press CTRL+C to quit
Step 4 Open in Browser

What to do: View the web interface

  1. Open your web browser (Chrome, Firefox, Edge, etc.)
  2. Go to http://localhost:5000
  3. You should see this website!
Step 5 Run Analysis

What to do: Execute the analysis

  1. Go to Run Analysis tab
  2. Click "Run All" button
  3. Wait for analysis to complete (~5-10 minutes)
  4. Check output log for completion message
Timing: Analysis takes 5-10 minutes total
Step 6 Explore Results

What to do: View your analysis results

  • Explore Data - Browse raw and cleaned data
  • Visualizations - View 6 professional charts
  • Insights - Read business recommendations
  • Documentation - Access complete guides
Setup Checklist
Troubleshooting
Web page won't load?

Make sure Flask app is running and you're using http://localhost:5000 (not https://)

Package installation error?

Try: pip install --upgrade pip first

File not found error?

Check that OnlineRetail.csv is in data/ folder
