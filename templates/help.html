Follow these steps:

Steps:

  1. Click link above
  2. Click "Download" button
  3. Extract ZIP file

Try these solutions:

  • Use http://localhost:5000 (NOT https://)
  • Check Flask is running in terminal
  • Wait 10-15 seconds and refresh (Ctrl+R)
  • Try different browser (Chrome, Firefox, Edge)
  • Restart Flask: Ctrl+C then python app.py

Expected timing:

  • Step 1 (Cleaning): 2-3 minutes
  • Step 2 (Analysis): 2-3 minutes
  • Step 3 (Statistics): 1-2 minutes
  • Total (All): 8-10 minutes

To speed up:

  • Close other programs
  • Close browser tabs
  • Run analysis alone (no other activity)
